## 这是什么

每个玩家出生时是一台**自由设定外观的变形金刚**。等级不靠经验值，由**变形齿轮**决定；传说中的**八个神秘变形齿轮**（潮涌、矿石、大地、森林、烈火、光明、黑暗、神秘）散落在世界各处。这个世界最软的东西是泥巴，最硬的东西是**代码**——一种黑色矿石，周围有绿色的字母连成线、纵横交错。

512×512 的主世界上有三块大陆：**初始城镇**（地下藏着驱动整个世界的地下之城）、**作者小岛**（创世神的木屋商店）、**收服大陆**（打败的怪物自动变成宠物）。用金子打开传送门去**暗黑地狱**，穿过丛林神殿踏入**终极之地**，在深海挑战海底守卫者，仰头找天上巡飞的**鲲鹏**，向千层塔顶的**地狱魔龙**发起冲锋——集齐八个神秘齿轮后，回到驱动核心旁的作者神殿见证大结局。

## 怎么玩

```bash
npm install
npm run dev
```

打开 http://localhost:5173 即可。**键盘鼠标或纯触控板都可以玩**（F 挖/打、R 放/开箱、方向键转视角）。

| 按键 | 作用 |
|---|---|
| WASD / 空格 | 移动 / 跳跃与游泳 |
| 左键·长按 F | 挖方块 / 攻击 |
| 右键 / R | 放方块 / 开箱 / 交互 |
| T | 变形（机器人/小车/潜水艇/矿石装甲/全能形态） |
| Q C X Z | 大地震击 / 烈焰喷射 / 净化闪光 / 隐身 |
| B / E / V | 背包 / 对话·上船 / 切换视角 |
| H | 被困时作者救援回城 |

## 技术

- **自研体素引擎**：chunk 面剔除网格、经典顶点 AO、mipmap、渲染距离剔除
- **四个维度**：主世界 / 暗黑地狱 / 终极之地 / 千层塔竞技场，懒生成 + 独立氛围
- **天空系统**：日月东升西落、渐变天穹、星空、漂移方块云、昼夜循环
- **全程序化纹理**：34 种方块材质全部 Canvas 代码绘制
- **水体**：游泳/涉水计时/有限流动扩散/世界坐标水面动画
- **粒子特效**：对象池粒子、技能特效、震屏、打击反馈
- **1000 层无尽爬塔**、宠物收服、五级剑甲合成、localStorage 存档三代迁移链
