## 操作方法

| 按键 | 作用 |
|---|---|
| 鼠标 / 触控板 / 方向键 | 转动视角 |
| W A S D | 走路（在船上就是开船） |
| 空格 | 跳 / 在水里往上游 |
| **按住左键 或 按住 F** | 挖方块（对准怪物按一下是攻击） |
| 右键 或 **R** | 放方块 / 开箱子 / 睡床 |
| **T** | 变形（形态由装备的齿轮决定） |
| **B** | 打开背包：物品 · 齿轮装备 · 合成 · 宠物 |
| **V** | 切换视角：第一视角 → 第三视角 → 第二视角（正面） |
| E | 对话 / 逛商店 / 上下船 / 传送台 / 进地下城 |
| **G 或 触屏 🔫** | 发射远程炮（拿到红眼睛后变成激光！） |
| **Q / C / X / Z** | 齿轮技能：地震 / 喷火 / 净化闪光 / 隐身 |
| **H** | 被困/迷路时按一下，作者把你接回出生点（主世界可用） |
| 数字键 1~5 / 双指滚动 | 选方块 |

> 🖐️ **触控板玩家**：方向键转视角、F 挖和打、R 放和开箱，完全不用鼠标键。

## 🆕 v4.0：初始城镇变成六区大岛！（你画的地图！）

出生的地方现在是一座**六个区域的大岛**，出城门就能去探险：

- 🏰 **中央城市**（你的家）——城墙围起来、四个大门、十字大道、作者之塔在正中。里面有 22 座房子、市集摊位、会走来走去的居民、还有四种商店（食品/建材/战斗装备/船和床）。**城里绝对安全，怪物进不来！** 地下之城的入口就在城里的广场。
- 🎋 **竹林**（城西）——高高的竹子和很多大熊猫。中间有**三层竹林神殿**：一层是竹子做身子、熊猫做脑袋的大图腾；二层住着 Boss**远古熊猫**（打败它得到**熊猫宠物**）；三层全是宝物。竹林深处藏着**神秘图腾**宝箱（装上伤害翻倍！）。
- 🗿 **巨石阵**（西北高地）——很多石柱，中央往下挖一百格，三块诡异木板下面是大地齿轮。
- ⛰️ **矿石群山**（城北）——五座大山，挖开能得钻石、红宝石、蓝宝石、金子、代码矿石，运气好还能挖到**神秘图腾**！
- 👻 **鬼城遗址**（城东）——一片废墟，有烟有火有撕裂声，出没**变异坦克**和**变异装甲车**，天上还盘旋一条会喷火焰光柱的**红色邪恶巨龙**！
- 🌲 **森林**（城南）——很密的树林，中间一棵超高的古树，还有**两层森林神殿**：二层的大箱子里是**红眼睛**——按 G 发射红色激光，一下 100 伤害！
