## 怎么打开游戏

请大人帮忙在这个文件夹里运行：

```bash
npm run dev
```

然后用浏览器打开 **http://localhost:5173** 就可以玩啦。

## 操作方法

| 按键 | 作用 |
|---|---|
| 鼠标 / 触控板 / 方向键 | 转动视角 |
| W A S D | 走路（在船上就是开船） |
| 空格 | 跳 / 在水里往上游 |
| **按住左键 或 按住 F** | 挖方块（对准怪物按一下是攻击） |
| 右键 或 **R** | 放方块 / 开箱子 / 睡床 |
| **T** | 变形（形态由装备的齿轮决定） |
| **B** | 打开背包：物品 · 齿轮装备 · 合成 · 宠物 |
| **V** | 切换视角：第一视角 → 第三视角 → 第二视角（正面） |
| E | 对话 / 逛商店 / 上下船 / 传送台 |
| **Q / C / X / Z** | 齿轮技能：地震 / 喷火 / 净化闪光 / 隐身 |
| **H** | 被困/迷路时按一下，作者把你接回出生点（主世界可用） |
| 数字键 1~5 / 双指滚动 | 选方块 |

> 🖐️ **触控板玩家**：方向键转视角、F 挖和打、R 放和开箱，完全不用鼠标键。

## 新世界有什么

- 🌊 **真正的海洋**：可以游泳！但普通情况下只能涉水十分钟，泡太久会受伤
- 🐦 **鲲鹏**：深海上空的巨大生物（又是鱼又是鸟）！打败它掉**鹏之药水**（喝了水里行动自如）和**鲲鹏之翼**（二段跳+滑翔）
- 🏝 **作者小岛**：木屋商店卖水果、海货、床、船（床晚上睡觉直接到天亮，船开着比游泳快）
- 🐾 **收服大陆**：打败的怪物自动变宠物，怪越强宠越强！蟹类还能骑！
- 🔥 **暗黑地狱**：城镇南滩的大门，门顶放一块金子就开（金子会全部消耗，一次只够一人进返！）——森林齿轮在岩浆湖中央树顶，烈火齿轮要勇敢走进火海正中央
- 🌌 **终极之地**：收服大陆丛林神殿的传送门——怪物靠悬空符号回血，先打碎符号！小心虚空！
- ⛏ **地下之城**：城镇地下的地核之城，地下族人守护着驱动核心，光明齿轮就在核心边上
- 🗼 **千层塔**：楼梯拆了！大厅传送台直达，打到哪层传到哪层，一路爬到第 1000 层的**地狱魔龙**（作者打半天才掉一滴血的那位）
- 🔨 **合成**：木剑到**代码神剑**（最硬的代码矿石打造！）五级剑甲
- 💠 **八大神秘变形齿轮**全部就位——集齐后回地下之城的作者神殿，有大结局等你！
